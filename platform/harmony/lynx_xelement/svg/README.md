Lynx XElement SVG
====

Lynx XElement SVG for the harmony platform.

## Overview

Lynx XElement SVG provide `<svg>` on harmony.

## Installation

```bash
ohpm install @lynx/xelement-svg
```

## How to use

You can add dependency in oh-package.json5 like this:

```json5
{
  "dependencies": {
    "@lynx/xelement_svg": "0.0.1-alpha.1",
  }
}
```
